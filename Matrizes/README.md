# Trabalho de Grafos e Matrizes  
**Matemática Discreta – Análise de Grafos**

Este projeto implementa matrizes e grafos utilizando o dataset fornecido pelo professor.  
Inclui:

- Matriz de Incidência  
- Matriz de Similaridade  
- Grafo de Incidência  
- Grafo de Similaridade  
- Arquivos CSV e PNG gerados automaticamente

A Matriz e o Grafo de Coocorrência serão feitos por outro integrante do grupo.

## Estrutura da Pasta
```
/Trabalho-Grafos/
│   Dataset.txt
│   trabalho_grafos.py
│   README.md
```

## Instalação das Bibliotecas

Instale as dependências com:

```bash
pip install pandas
pip install numpy
pip install networkx
pip install matplotlib
```

## Para que serve cada biblioteca?

- **pandas** – leitura e manipulação de dados  
- **numpy** – multiplicação de matrizes  
- **networkx** – criação e análise de grafos  
- **matplotlib** – geração das imagens PNG  

## Como Executar o Projeto

```bash
python trabalho_grafos.py
```

O script gera automaticamente:

```
matriz_incidencia.csv
matriz_similaridade.csv
grafo_incidencia.png
grafo_similaridade.png
```

## Observação Importante

O trabalho exige 3 matrizes e 3 grafos:

- Incidência → este integrante  
- Similaridade → este integrante  
- Coocorrência → outro integrante
